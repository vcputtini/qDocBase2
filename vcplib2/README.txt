Instrucoes para uso desta biblioteca

NOTAS IMPORTANTES:======================================================

Manter sempre atualizados os arquivos de inclusao nas pastas "includes"
para que a compilacao dos programas nao tenham problemas.
========================================================================

1) LINUX

    Cria um diretorio em /usr/include com o nome vcplib2
    e copiar todos os arquivos .h para ele.

    Copiar a biblioteca e os links para o /lib64


2) WINDOWS

    Criar uma pasta em C:/Qt/5.x.y/5.x.y/include com o nome vcplib2
    e copiar todos os arquivos .h para ele.

    Copiar a dll para a pasta de compilacao/instalacao da aplicacao

    A compilacao *DEVE* ser em 32bits e ser utilizado o MySQL ou MariaDB
    tambem de 32BITS
